## installation

>linux ubuntu
1. Download the file gox.xx.x.linux-amd64.tar.gz
2. sudo tar -C /usr/local -xzf gox.xx.x.linux-amd64.tar.gz
3. ls /usr/local/go
4. vi ~/.profile
5. in the end write: export PATH=$PATH:/usr/local/go/bin
6. go version
